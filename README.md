# مهامي اليومية PWA v5

نسخة مربوطة مع OneSignal Web Push باستخدام App ID المقدم.

ارفع الملفات كلها إلى GitHub Pages مكان الملفات القديمة، ثم افتح التطبيق من أيقونة الشاشة الرئيسية على الآيفون وفعّل الإشعارات.

ملاحظة: هذه النسخة تجهز الاشتراك في Web Push. لإرسال إشعار عند موعد كل مهمة والشاشة مغلقة نحتاج مرحلة لاحقة لإرسال الإشعار من OneSignal Dashboard أو API عبر خادم آمن، لأن REST API Key لا يجوز وضعه داخل HTML.
