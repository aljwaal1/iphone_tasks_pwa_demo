# مهامي اليومية PWA v7

نسخة فحص إشعارات OneSignal للآيفون.

## طريقة الاستخدام
1. ارفع كل الملفات مكان القديمة في GitHub.
2. انتظر دقيقة.
3. احذف أيقونة التطبيق القديمة من الآيفون.
4. افتح الرابط من Safari واسحب تحديث.
5. أضف التطبيق مرة أخرى إلى الشاشة الرئيسية.
6. افتحه من الأيقونة واضغط: فحص الإشعارات أو تفعيل إشعارات قفل الشاشة.
