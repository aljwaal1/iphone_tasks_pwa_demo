# مهامي اليومية PWA v13

إصلاح خاص باستضافة GitHub Pages داخل مجلد المشروع /iphone_tasks_pwa_demo/.

آخر التعديلات (v13):
- تذكير يومي تلقائي الساعة 10 صباحًا بعدد المهام غير المنجزة.
- ألوان أولوية واضحة وصلبة (أحمر/أصفر/فيروزي) بدل الشفافية السابقة.
- حذف ملف `sw.js` (كان غير مفعّل أبدًا) ودمج التخزين المؤقت (Offline Cache) داخل `OneSignalSDKWorker.js` نفسه — وهو الملف الفعلي المسجَّل، فصار التخزين المؤقت يعمل بالفعل الآن.

تعديلات سابقة (v9):
- ضبط مسار OneSignal Service Worker بشكل مطلق.
- ضبط نطاق Service Worker على /iphone_tasks_pwa_demo/.
- ضبط manifest start_url و scope لنفس المسار.

ارفع الملفات كلها مكان القديمة. بما أن `sw.js` حُذف، يُفضّل حذف أيقونة الآيفون من الشاشة الرئيسية وإضافتها من جديد من Safari حتى يُسجَّل Service Worker الجديد بشكل نظيف.
