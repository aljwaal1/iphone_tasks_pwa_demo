# مهامي اليومية PWA v4

نسخة مطورة للآيفون تعمل كتطبيق PWA.

## أهم التعديل في v4
- فصل التاريخ عن الوقت.
- حقل وقت واضح بالساعة والدقيقة.
- يمكن اختيار وقت خلال اليوم بسهولة.
- إذا وضعت الوقت فقط، يتم اعتماد تاريخ اليوم تلقائيًا.
- أزرار سريعة: بعد 30 دقيقة، بعد ساعة، اليوم 8 مساءً.

ارفع الملفات إلى GitHub Pages مكان الملفات القديمة.
